ATOMIA — Dove l'Energia Diventa Vita

Ridefinire l'Energia, Riimmaginare il Domani

ATOMIA è un sito web concept che esplora una città futuristica alimentata da energia nucleare illimitata. Il progetto è una landing page visivamente immersiva, costruita interamente in HTML, CSS e JavaScript vanilla, senza framework o dipendenze esterne.

Caratteristiche:

Cursore custom — cursore personalizzato con anello che segue il mouse (solo desktop)
Scroll reveal — gli elementi appaiono con animazione al momento dello scroll
Effetti parallax — profondità visiva durante lo scorrimento (solo desktop)
SVG animati — alberi energetici e cristalli disegnati in SVG con glow e pulsazioni
Design completamente responsive — ottimizzato per mobile, tablet e desktop

Tecnologie

HTML5
CSS3 — animazioni, variabili CSS, media query, grid e flexbox
JavaScript — Canvas API per le particelle, IntersectionObserver per lo scroll reveal
Google Fonts — Syne (display) + Outfit (body)

Note di design

Palette colori: #020810 (sfondo), #00d4ff (cyan), #00ffd0 (teal)
Gli effetti parallax e il cursore custom sono disattivati automaticamente su dispositivi touch per garantire performance ottimali
Le particelle sono ridotte a 40 su mobile (80 su desktop) per non appesantire la batteria
